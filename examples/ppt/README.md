# PPT智能生成示例

本目录包含PPT智能生成功能的示例和说明。

## 核心流程

```
用户输入 → 意图解析 → 主题搜索 → 素材获取 → 智能排版 → 成品输出 → 反馈学习
     ↑___________________________________________________________↓
```

## 示例场景

### 场景1：科技风项目介绍PPT

**输入**：
```
科技风项目介绍PPT，包含背景、解决方案、技术优势、团队介绍等内容
```

**处理流程**：
1. 意图解析：识别主题="项目介绍"，风格="科技风"
2. 结构建议：封面→目录→背景→解决方案→技术优势→团队→封底
3. 素材搜索：搜索科技风格背景图、技术相关图标
4. 智能排版：根据内容类型选择合适布局
5. 输出成品：生成可编辑的.pptx文件

### 场景2：商务工作汇报PPT

**输入**：
```
商务风格的年度工作汇报，10页左右
```

**处理流程**：
1. 意图解析：识别主题="工作汇报"，风格="商务风"
2. 页数设定：10页
3. 结构建议：封面→目录→概述→完成情况→问题分析→下步计划→总结→封底
4. 素材搜索：商务风格图表、数据可视化元素
5. 智能排版：数据页使用图表聚焦布局

### 场景3：产品发布PPT

**输入**：
```
创意风格的产品发布会PPT，突出产品特点
```

**处理流程**：
1. 意图解析：识别主题="产品发布"，风格="创意风"
2. 结构建议：封面→痛点→产品介绍→功能特点→用户价值→市场前景→封底
3. 素材搜索：创意风格配图、产品展示元素
4. 智能排版：使用大图背景、创意布局

## 布局类型说明

| 布局类型 | 适用场景 | 特点 |
|---------|---------|------|
| full_image | 封面页 | 全图背景+居中标题 |
| left_nav | 目录页 | 左侧导航+右侧内容 |
| text_image_right | 内容页 | 文字左+图片右 |
| text_image_left | 内容页 | 图片左+文字右 |
| two_column | 对比页 | 左右双栏对比 |
| three_column | 要点页 | 三栏并列展示 |
| grid_2x2 | 案例页 | 2×2网格布局 |
| chart_focus | 数据页 | 图表居中+说明文字 |
| timeline_horizontal | 时间线 | 水平时间轴 |
| team_grid | 团队页 | 团队成员网格 |
| contact_info | 封底页 | 联系方式+二维码 |

## 风格说明

### 科技风 (tech)
- 主色调：蓝色系 (#2196F3)
- 特点：渐变背景、科技元素、简洁线条
- 适用：互联网、科技、创新项目

### 商务风 (business)
- 主色调：橙色/深蓝 (#FF5722 / #1A237E)
- 特点：正式、专业、数据驱动
- 适用：企业汇报、商业计划、投资路演

### 极简风 (minimal)
- 主色调：灰色系 (#607D8B)
- 特点：大量留白、简洁排版、突出内容
- 适用：设计展示、艺术类、高端品牌

### 创意风 (creative)
- 主色调：紫色/多彩 (#9C27B0)
- 特点：大胆配色、创意布局、视觉冲击
- 适用：创意产品、品牌发布、艺术活动

### 学术风 (academic)
- 主色调：绿色系 (#4CAF50)
- 特点：严谨、清晰、图表丰富
- 适用：学术报告、教学课件、研究成果

## 学习机制

### 素材质量评分
- 使用频率：被选用的次数
- 用户评分：1-5星评价
- 保留率：生成后未被删除的比例

### 用户偏好学习
- 风格偏好：统计用户常用风格
- 颜色偏好：分析用户选择的配色
- 布局偏好：记录用户偏好的页面布局

### 自动优化
- 热门素材推荐：根据全局使用统计
- 个性化排序：根据用户历史偏好
- 素材淘汰：长期未使用自动归档

## API使用示例

```python
from ppt.backend import PPTPipeline, PPTStyle

# 创建生成流水线
pipeline = PPTPipeline()

# 基础生成
result = pipeline.run("科技风项目介绍PPT")

# 带用户ID生成（启用个性化）
result = pipeline.run("商务汇报PPT", user_id="user123")

# 检查结果
if result.success:
    print(f"PPT已生成：{result.file_path}")
    print(f"共 {len(result.pages)} 页")
else:
    print(f"生成失败：{result.error}")
```

## 反馈与迭代

生成完成后，用户可以：
1. **单页调整**：右键某页选择"AI重排"
2. **风格切换**：保留内容，更换风格和素材
3. **内容扩充**：添加新页面，自动搜索相关素材
4. **评分反馈**：对素材和整体效果评分
