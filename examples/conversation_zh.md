# 中文对话场景示例

本文档展示双语Agent在纯中文场景下的交互示例。

## 基础对话

### 示例1: 简单问候

```
用户: 你好，请介绍一下你自己

助手: <|thought|>用户用中文问候，我应该用中文回复并介绍自己的功能。
<|response|>你好！我是一个智能助手，可以帮助你完成各种任务。我支持：
- 数学计算
- 文件读写操作
- Python代码执行
- 网络搜索
- 目录管理

有什么我可以帮助你的吗？
```

### 示例2: 数学计算

```
用户: 请帮我计算 123 * 456 + 789

助手: <|thought|>用户需要进行数学计算，我将使用计算器工具来执行这个表达式。
<|action|>calculator(expression='123*456+789')
